💻 Windows – Kommandoer

CMD / PowerShell – Nettverk
ipconfig              # Vis nettverkskonfigurasjon
ipconfig /all         # Detaljert nettverksinformasjon
ping <IP eller navn>  # Test tilkobling
tracert google.com    # Sporingsrute
ipconfig /flushdns    # Tøm DNS-cache
netstat -an           # Se aktive nettverksforbindelser
CMD – Brukeradministrasjon (lokal)
net user                             # Vis alle brukere
net user <navn> <passord> /add       # Opprett bruker
net user <navn> /delete              # Slett bruker
net localgroup <gruppe> <bruker> /add # Legg i gruppe
net localgroup Administrators /add   # Legg i Admin-gruppen
PowerShell – Active Directory
Get-ADUser -Filter *                              # Alle AD-brukere
New-ADUser -Name "Navn" -SamAccountName "bruker"  # Opprett AD-bruker
New-ADGroup -Name "Gruppe" -GroupScope Global     # Opprett gruppe
Add-ADGroupMember -Identity "Gruppe" -Members "bruker" # Legg til i gruppe
Disable-ADAccount -Identity "bruker"              # Deaktiver konto
Enable-ADAccount -Identity "bruker"               # Aktiver konto

🐧 Ubuntu – Kommandoer

Bash – Nettverk
ip addr               # Vis nettverkskonfigurasjon
ip route              # Vis rutetabell
ping -c 4 <IP>        # Ping (4 pakker)
nslookup google.com   # DNS-oppslag
traceroute google.com # Sporingsrute
Bash – Brukeradministrasjon
sudo adduser <navn>              # Opprett bruker (interaktivt)
sudo deluser --remove-home <n>  # Slett bruker og hjemmappe
sudo passwd <navn>              # Endre passord
sudo usermod -aG sudo <bruker> # Gi sudo-rettigheter
cat /etc/passwd                 # Vis alle brukere
groups <brukernavn>             # Vis gruppemedlemskap
su - <brukernavn>               # Bytt til bruker
whoami                          # Hvem er du?
